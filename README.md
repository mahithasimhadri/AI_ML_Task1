# Task 1: Data Cleaning and Preprocessing

## Objective

The objective of this task is to clean and preprocess the Titanic dataset and prepare it for machine learning.

## Dataset

Titanic Dataset

## Steps Performed

1. Loaded the dataset using Pandas.
2. Explored dataset structure and data types.
3. Identified missing values.
4. Filled missing values in Age using median.
5. Filled missing values in Embarked using mode.
6. Removed Cabin column due to excessive missing values.
7. Encoded categorical variables using Label Encoding.
8. Visualized outliers using boxplots.
9. Removed outliers using the IQR method.
10. Standardized Age and Fare using StandardScaler.
11. Saved the cleaned dataset.

## Libraries Used

* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Output

A cleaned and preprocessed Titanic dataset ready for machine learning applications.
